Microcontrolador Wi‑Fi + Bluetooth

ESP32

El ESP32 es una familia de SoC (System‑on‑Chip) de Espressif Systems ideal para IoT, automatización y prototipado rápido. Integra conectividad inalámbrica, buena potencia de cómputo y un ecosistema enorme.

¿Qué es el ESP32?

Es un microcontrolador moderno pensado para proyectos conectados. Dependiendo del módulo/placa, suele incluir antena, regulador y pines listos para usar.

Ver información
  • SoC con Wi‑Fi 2.4 GHz y Bluetooth (clásico/BLE según variante).
  • Soporta múltiples periféricos para sensores, actuadores y comunicación.
  • Ideal para domótica, telemetría, estaciones meteorológicas, wearables y más.

Conectividad

La conectividad es uno de sus puntos fuertes: perfecto para proyectos IoT sin módulos externos.

Ver información
  • Wi‑Fi: modo estación/AP, redes domésticas y hotspots.
  • Bluetooth: BLE para sensores, beacons, apps móviles y comunicación eficiente.
  • Redes: HTTP/HTTPS, MQTT, WebSockets y más (según librerías/firmware).

Periféricos y pines

Ofrece muchas opciones de entrada/salida para conectar módulos y sensores.

Ver información
  • GPIO, PWM, ADC (entradas analógicas) y DAC (en algunos modelos).
  • UART, I2C, SPI para pantallas, memorias, sensores, etc.
  • Touch capacitivo en varios pines (según el chip).

Nota: la disponibilidad real de pines depende de la placa (DevKit, WROOM, WROVER, etc.).

Programación

Se puede programar con varias herramientas populares, según tu estilo y necesidades.

Ver información
  • Arduino IDE: fácil para comenzar y con muchas librerías.
  • ESP-IDF: framework oficial con más control y opciones avanzadas.
  • MicroPython: ideal para prototipos rápidos con Python.

Ideas de proyectos

Control por Wi‑Fi Sensores + MQTT Panel web BLE + App móvil Automatización

Un ESP32 puede alojar un servidor web para controlar LEDs/relevadores, enviar datos de sensores a la nube, registrar mediciones en una tarjeta SD, o comunicarse con una app móvil por BLE.
